### Normal Flow

```
[Producer]                      [Broker]                      [Worker]
    |                              |                              |
    |--(1) Create Task------------->|                              |
    |   task_id = uuid             |                              |
    |                              |                              |
    |--(2) Enqueue----------------->|                              |
    |   queue: default             |                              |
    |   priority: 5                |                              |
    |                              |                              |
    |                              |<-(3) Dequeue-----------------|
    |                              |                              |
    |                              |<-(4) ACK---------------------|
    |                              |                              |
    |<-(5) Poll for Result---------|                              |
    |                              |                              |
    |                              |<-(6) Store Result------------|
    |                              |                              |
    |<-(7) Get Result--------------|                              |
    |   result.get(timeout=10)    |                              |
    |                              |                              |
```

```

    |                              |                              |
    |--(1) Execute Task------------>|                              |
    |                              |                              |
    |--(2) Task Fails------------->|                              |
    |   ConnectionError            |                              |
    |                              |                              |
    |                              |<-(3) Check Retry Policy------|
    |   retries=3                  |                              |
    |                              |                              |
    |                              |<-(4) Calculate Backoff-------|

    |                              |                              |
```
